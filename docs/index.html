EO dialect* editor

based on Codemirror 6 | permalink to this snippet | GitHub repo | compare with EO snippets | EO repo

Action Keybinding
Indent selection Tab
Dedent (a selected block) Shift+Tab
Underline selection Ctrl+H
Insert EOL Enter
Undo the last text action Ctrl+Z
Diagnostic Meaning
A red triangle A position of a parsing error
Red wavy underlines (possibly) a part of an incorrect expression
If toggled, full Lezer parse tree is in your browser's console

* The only syntactic difference from EO is that for an attribute (or an object), its name should always precede its value when reading left-to-right, top-to-bottom [discussion]
